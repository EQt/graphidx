template<typename int_ = int>
void prufer2edges(const int n, const int_ *prufer, int_ *head, int_ *tail)¶

Calculate the tree edges from a Prüfer sequence.

Runtime: O(n).

Warning

Need the following pre-conditions

  • len(prufer) == n-2

  • len(head) >= n-1

  • len(tail) >= n-1

Parameters

  • [in] n: Number of nodes in the tree

  • [in] prufer: Prüfer sequence;

  • [out] head: Output edges’ head

  • [out] tail: Output edges’ tail

References

Wang, Xiaodong, Lei Wang, and Yingjie Wu. “An optimal algorithm for Prufer codes.” Journal of Software Engineering and Applications* 2.02 (2009): 111. https://doi.org/10.4236/jsea.2009.22016

Binary van Emde Boas Layout from an Inorder traverse (sorted nodes).

Copied and adapted from Frederik Rønn: “Cache-Oblivious Searching and Sorting”, Master’s Thesis, Department of Computer Science University of Copenhagen, 2003. (Page 76)

Parameters

  • [out] a_out: Array to store the output.

  • [in] a_in: Binary inorder (as produced by binary_inorder)

  • [in] height: Height of the complete binary tree.

  • [in] step: Distance between siblings on the lowest level. Start with step == 1.
