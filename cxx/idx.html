Graph Indexes¶

The unique feature of this graph framework is to compute indices to be able to access certain elements in constant time. The index computation is always based on counting sort to group certain elements together by an index:

Functions

template<typename int_ = int>
void groupby(std::vector<int_> &value, std::vector<int_> &index, const size_t n, const int_ *parent, const int_ root = -1)¶

Group the elements in parent, stored in value, group index index.

If root >= 0 treat it as the root of the tree given by parent. The root node if >= 0 will be stored at index[0].

Algorithm: Counting Sort (linear runtime)
