GraphIdx¶

This is the documentation of the C++ part of GraphIdx. For the Julia documentation see here.

The library is header only, so setting the include path apropriately suffices.

Edges¶

If not stated otherwise, the graphs considered here are undirected. Consequently, an index containing the neighbors for each node uses \(2|E| + |V|\) many integers.

Node Type¶

Within this library, nodes are represented by integers. For being able to encode more than 2G nodes, mostly the used integer types ared templated with default to int (this often reduces the memory bandwidth compared to int64_t).

Furtheron, we decided to use signed integers such that the first bit may encode additional information.
